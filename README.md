Voice Assistant Using Python
📌 Objective

The objective of this project is to create a simple voice assistant using Python that can recognize voice commands and respond with basic actions such as telling the time, searching the web, or answering simple questions.

🛠 Tools & Technologies Used

Python

SpeechRecognition library

Pyttsx3 (Text-to-Speech)

Webbrowser module

📂 Project Description

This project is a basic voice assistant that listens to user voice commands through the microphone and performs tasks accordingly. It converts speech into text and processes the command to provide a response.

⚙️ Steps Performed

Installed required Python libraries.

Created a Python script for voice recognition.

Implemented text-to-speech functionality.

Added commands like opening websites and telling time.

Tested the voice assistant.

▶️ How to Run the Project

Install required libraries:

pip install SpeechRecognition pyttsx3

Run the Python file:

python voice_assistant.py

Speak commands through the microphone.

📊 Outcome

The voice assistant successfully recognizes voice commands and performs the assigned tasks, demonstrating the use of speech recognition and automation in Python.
